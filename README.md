## Included Tools

### 1. Simple Click Test
Located in `src/simple-click-test.html`.
- **Zero Dependencies**: Just a single HTML file.
- **Offline Ready**: Works without internet.
- **Features**: Counts clicks, tracks high scores (saved locally).

### 2. Mouse Maintenance Guide
Located in `docs/mouse-maintenance.md`.
- Comprehensive guide on fixing common mouse problems.
- How to clean optical sensors.
- How to fix double-clicking switches using contact cleaner.

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/mrliang-github/doubleclicktestoffpage.git
   ```
2. Open `src/simple-click-test.html` in any web browser.
